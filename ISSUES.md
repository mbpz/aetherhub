## 🚀 Issue创建步骤

### 方式1: 使用GitHub CLI (推荐)

```bash
# 1. 安装GitHub CLI
brew install gh

# 2. 登录
gh auth login

# 3. 运行创建脚本
cd aetherhub
python3 scripts/create_test_issues.py
```

### 方式2: 手动创建

每个测试用例的Issue内容在 `scripts/create_test_issues.py` 中定义。

## 📝 Issue模板

每个Issue包含：
- **标题**: `[TC-XX-YY] 测试用例名称`
- **标签**: `test-case`, `priority:P0/P1/P2`
- **内容**:
  - 测试用例信息（PRD、优先级、类型、状态）
  - 前置条件
  - 测试步骤
  - 预期结果
  - 验收标准
  - 相关文档链接

## 🔄 Issue状态追踪

- **待测试**: Issue创建后，测试用例编写中
- **测试通过**: 所有验收标准已满足
- **测试失败**: 需要修复问题
- **已关闭**: Issue已解决
